Breadcrumb Diagnostic for Átria Veículos

Execute this in the browser console on a vehicle detail page:

// PASSO 1 - Confirmar quantos breadcrumbs existem e onde estão
const breadcrumbDiagnostic = () => {
    const result = [...document.querySelectorAll('nav[aria-label="breadcrumb"]')].map((n,i)=>({
        idx: i,
        text: n.innerText.trim().replace(/\s+/g,' '),
        inHeader: !!n.closest('header, .boxcar-header'),
        path: (el => { 
            const p=[]; 
            while(el && el!==document.body){ 
                p.unshift(el.tagName.toLowerCase()+(el.id?`#${el.id}`:'')+(el.className?'.'+[...el.classList].join('.'):'') ); 
                el=el.parentElement; 
            } 
            return p.join(' > '); 
        })(n)
    }));
    
    console.log('🔍 BREADCRUMB DIAGNOSTIC RESULTS:', result);
    
    if (result.length === 0) {
        console.log('❌ NO BREADCRUMBS FOUND - VehicleBreadcrumb may not be rendering');
    } else if (result.length === 1) {
        const breadcrumb = result[0];
        if (breadcrumb.inHeader) {
            console.log('⚠️ SINGLE BREADCRUMB IN HEADER - Should be in main content');
        } else {
            console.log('✅ SINGLE BREADCRUMB CORRECTLY POSITIONED - Check for spacing issues');
        }
    } else {
        console.log('❌ MULTIPLE BREADCRUMBS FOUND - Duplicate breadcrumb issue');
        result.forEach((bc, i) => {
            console.log(`Breadcrumb ${i+1}:`, bc.inHeader ? 'IN HEADER' : 'IN MAIN', '→', bc.text);
        });
    }
    
    return result;
};

breadcrumbDiagnostic();

Instructions:

  1. Go to any vehicle detail page (e.g., /carros/hyundai/hb20/2018-AMM26)
  2. Open browser console (F12 → Console)
  3. Paste and execute the code above
  4. Report back the console output

Expected Results:

  • 0 breadcrumbs: VehicleBreadcrumb not rendering
  • 1 breadcrumb + inHeader: false: ✅ Correct (check spacing)
  • 1 breadcrumb + inHeader: true: ⚠️ Wrong position
  • 2+ breadcrumbs: ❌ Duplicates exist
